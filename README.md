# Images Folder

This folder contains all graphics and media assets for the NeuroType website.

## Contents

- **nt-logo.svg** – Main site logo with atom design, used in branding
- **favicon.svg** – Small icon displayed in browser tabs (can be converted to .ico)

## Next Steps

1. **Add favicon to HTML** – Link the favicon in your `<head>` section:
   ```html
   <link rel="icon" type="image/svg+xml" href="images/favicon.svg">
   ```

2. **Use logo in pages** – Reference `nt-logo.svg` in headers or footers if desired

3. **Add screenshots** – Capture your typing race in action and add to this folder for your README

4. **Optimize images** – For production, convert SVGs to PNG if needed for better browser support

## Color Scheme

All graphics use the site's palette:
- Primary: #b7b18f (tan/brown)
- Secondary: #d6d0b8 (lighter tan)
- Accent: #4CAF50 (green)
- Dark: #333
